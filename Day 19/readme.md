# Football Intelligence Hub

## Overview

Football Intelligence Hub is an interactive football analytics, education, and personality assessment experience designed to combine football knowledge, tournament forecasting, and player-style personality analysis into a single engaging journey.

The system guides users through three stages:

1. Knowledge Level Assessment
2. FIFA World Cup 2026 Prediction Analysis
3. Football IQ & Personality Evaluation

The final result is a personalized Football Intelligence Profile.

---

# Features

## Stage 0 — Knowledge Level Check

The experience begins by assessing the user's football familiarity.

### User Levels

* Level A — I know almost nothing about football
* Level B — I know the basic rules and recognize a few famous players
* Level C — I watch some matches and follow major tournaments occasionally
* Level D — I regularly follow football leagues, clubs, and international competitions
* Level E — I actively follow football, player statistics, tactics, transfers, and major tournaments

### Purpose

The selected level adjusts:

* Explanation depth
* Football terminology
* Tactical complexity
* Historical references
* Educational examples

No scoring is performed during this stage.

# Stage 2 — Football IQ Quiz

Generate 4–5 multiple-choice questions.

Questions should include a mixture of:

* Beginner
* Intermediate
* Advanced

difficulty levels.

## Example Topics

### Beginner

* Rules of football
* World Cup history
* Famous players

### Intermediate

* Formations
* Tactical concepts
* International competitions

### Advanced

* Pressing systems
* Positional play
* Statistical analysis
* Tournament strategy

## Scoring

Calculate:

Football Awareness Score (0–100)

### Classification Levels

| Score  | Classification      |
| ------ | ------------------- |
| 0–20   | Beginner Fan        |
| 21–40  | Casual Viewer       |
| 41–60  | Football Follower   |
| 61–80  | Football Enthusiast |
| 81–100 | Football Expert     |

## Feedback

Provide:

### Strongest Areas

Knowledge categories where the user performed best.

### Weakest Areas

Areas requiring improvement.

### Key Knowledge Gaps

Specific topics recommended for learning.

* Competitiveness
* Confidence
* Work Ethic
* Learning Style
* Decision Making

### Multiple Choice

Examples:

* Approach to competition

1–5 or 1–10
