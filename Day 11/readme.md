# Shivam Yadav Portfolio Website

A modern, responsive, and recruiter-friendly personal portfolio website built using **HTML**, **Tailwind CSS**, and **Vanilla JavaScript**.

## 🚀 Overview

This portfolio showcases the professional profile, technical skills, projects, achievements, certifications, and contact information of **Shivam Yadav**, a Computer Science graduate passionate about Software Development, Machine Learning, and Competitive Programming.

The website is designed with a modern UI, smooth animations, responsive layouts, and a professional personal branding approach suitable for recruiters and hiring managers.
